# Io Persona Health Dashboard (v1.0)

This dashboard is a **lightweight operational monitoring tool** for Io v1.3 + ICP v1.0.

Use it to log:

- Drift incidents
- Confusion patterns
- Mode mismatches
- Safety edge-cases
- Improvement ideas

Update it periodically (e.g., weekly).
